走进WordPress5.7.1。

在这个新版本中，WordPress将为您带来鲜活色彩。编辑器可以帮助您在一些您以前做不到的地方工作——至少在不涉及代码层面或者雇佣专业人员的情况下是如此。您最常用的控件——如更改字体大小——位于更多您需要它们的位置。布局的改变——例如全高图像——本应很简单，现在已经更加简单。

维护和安全更新

5.7.1版本修补了安全漏洞、修正了26个问题。 欲了解更多，参见发行注记。

编辑器现在更易于使用

更多位置的字体大小调整： 现在，字体大小控件将显示于“列表”和“代码”区块中需要它们的地方，不再需要跳转到另一个界面，字体调整一步到位！

可重用区块：多项增强功能使可重用区块更稳定且易于使用。现在，当您点击“更新”按钮时，它们会与文章一起自动保存。

插入器拖放：可将区块和区块模式从插入器直接拖到您的文章中。

您无需编写自定义代码即可完成更多工作

全高对齐：您是否想过让一个区块（如封面区块）可以填充整个窗口？现在终于可以了。

按钮区块：现在您可以选择垂直或水平布局。您还可以将按钮的宽度设置为预设的百分比。

社交图标区块：现在您可以更改图标的大小。

更简单的默认调色盘

这个新的精简调色盘将WordPress源代码中的所有颜色折叠成7种核心颜色和56种色调，这些颜色与白色或黑色都符合WCAG 2.0 AA推荐的对比度。

在每个范围内，从亮到暗的颜色在视觉上是一致的，就是说它们从白色开始，每一步都以相同的量变暗。

范围的一半对黑色的对比度在4.5以上，另一半对白色的对比度保持不变。

在默认的WordPress仪表盘配色方案中找到新的调色盘，并在构建主题、插件或任何其他组件时使用。更多详细信息请查阅颜色调色盘开发说明。

从HTTP到HTTPS一键搞定

从现在开始，可以一键式将站点从HTTP切换到HTTPS。当您进行切换时，WordPress会自动更新数据库的URL。再也无需寻找和猜测啦！

全新Robots API

全新的Robots API允许您在Robots meta标记中包含过滤器指令，默认情况下，该API将包含max-image-preview: large指令。这意味着搜索引擎可以显示更大的图片预览，可以增加您网站的流量（除非网站被标记为非公开）。

更新到jQuery 3.5.1后正在进行的清理。

多年来，jQuery帮助让事物以基本工具无法实现的方式在屏幕上移动——但此种情况一直在变更，jQuery也是如此。

在WordPress 5.7中，jQuery将变得更专注，带来更小的干扰，留下更少的控制台消息。

延迟加载您的iframe

现在让iframe延迟载入很简单。默认情况下，当iframe标签中宽度和高度属性值都被指定时，WordPress会向iframe标签添加一个loading="lazy"属性。
