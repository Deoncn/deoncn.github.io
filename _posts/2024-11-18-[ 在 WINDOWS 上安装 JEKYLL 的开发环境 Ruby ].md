# Main

1.  安装 <a href="https://rubyinstaller.org/downloads/archives/" target="blank">RubyInstallers+DEVKIT</a>.
2. 根据自己的 Jekyll 选择安装对应的版本
3. 安装完成后，gem install xxx Program.
4. 配置解释器，启动应用。
