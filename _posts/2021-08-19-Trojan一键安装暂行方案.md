```shell
bash <(curl -sL https://s.hijk.art/trojan.sh)
```

在终端（黑框框）输入上面的命令，按回车键，将出现如下操作菜单。如果菜单没出现，CentOS系统请输入 yum install -y curl，Ubuntu/Debian系统请输入 sudo apt install -y curl，然后再次运行上面的命令：
